Online пиццерия.

Пицца имеет название и определенный список ингредиентов.
Стоимость пиццы определяется исходя из списка ингредиентов и размера пиццы.
Заказ доставляется бесплатно, если стоимость заказа больше определенной суммы

Админ может добавлять/удалять пиццы, ингредиенты, изменять стоимость ингредиентов, изменять условия бесплатной доставки.
У него есть доступ ко всем таблицам. Он может просматривать все заказы, покупателей, их адреса доставки,
фильтруя по определенным критериям.

Зарегистрированный покупатель (user) может просматривать и заказывать пиццу, смотреть историю своих заказов,
может добавлять/удалять свои адреса доставки, менять информацию о себе,
при заказе пиццы может добавлять или удалять ингредиенты.

Незарегистрированный покупатель может только просматривать начальную страницу с выбором пицц. При попытке
заказа пиццы его беребрасывает на страницу регистрации.
